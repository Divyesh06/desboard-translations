# Deskboard Translations

This repository contains all the language packs used by Deskboard.

We welcome contributions to:
- Fix or improve existing translations  
- Add support for new languages  


## What is Deskboard?

Deskboard is a free app that turns your folders into visual boards.

Learn more: https://deskboard.geeke.app


## How to Contribute

1. Fork this repository  
2. Make your changes to the translation files  
3. Submit a pull request  

Once approved, the updated language pack will be available to all users.
